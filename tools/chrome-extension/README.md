# P-GATE Chrome Extension v0.1

Manifest V3の最小権限パイロット版。拡張アイコンを押した時だけ、現在のタブの商品タイトルまたは利用者が選択した文字を読み取る。閲覧履歴、Cookie、入力フォーム、ページ本文全体は取得しない。

## 初期機能

1. EC商品ページを開く。
2. 必要なら商品名や説明の一部を選択する。
3. P-GATEアイコンを押す。
4. サイドパネルの相談内容を確認し、同意する。
5. P-GATE PWAを開いて検索する。

質問はURLのクエリではなくフラグメントで渡す。フラグメントはHTTP要求へ送信されないため、静的配信サーバーのURLログへ質問を含めない。

## 権限

- `activeTab`: 利用者がアイコンを押した現在タブだけ
- `scripting`: 現在タブからタイトル・選択文字だけを取得
- `sidePanel`: 相談確認画面
- `storage`: P-GATE接続先URL

`host_permissions`は使用しない。外部コード、広告SDK、解析SDKも使用しない。
