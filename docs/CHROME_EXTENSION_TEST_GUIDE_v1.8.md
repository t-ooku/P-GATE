# Chrome拡張ローカル確認手順 v1.8

PWAのURLが確定した後、Chrome Web Store公開前に行う。

1. Chromeで`chrome://extensions`を開く。
2. 右上の「デベロッパー モード」をオンにする。
3. 「パッケージ化されていない拡張機能を読み込む」を押す。
4. 展開したソースの`tools/chrome-extension`フォルダを選ぶ。
5. P-GATE拡張の「詳細 → 拡張機能のオプション」を開く。
6. デプロイ済みPWAの`https://...` URLを保存する。
7. 商品ページを開き、商品名の一部を選択する。
8. ツールバーの拡張機能一覧からP-GATEを押す。
9. サイドパネルの文字を確認し、同意して「P-GATEで相談する」を押す。
10. PWAの相談欄に文字が入り、URL欄に質問本文が残らないことを確認する。

## 確認項目

- 通常の商品ページでは選択文字、未選択時はページタイトルが入る
- `chrome://`等では直接入力へフォールバックする
- 同意なしではPWAを開かない
- 2文字未満／200文字超を拒否する
- HTTP、認証情報付きURL、クエリ付きURLを接続先として拒否する
- 拡張の権限に「すべてのウェブサイト上のデータの読み取り」がない
- PWAへ移った後はTurnstileと利用同意が必要
